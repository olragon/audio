CBR ‚Äî Constant Bitrate

Every second uses the same bits. Valid: 8, 16, 24, 32, 40, 48, 64, 80, 96, 112, 128, 160, 192, 224, 256, 320.

VBR ‚Äî Variable Bitrate

Adjusts per complexity. V0 (best) ‚Üí V9 (worst). Different bits each second.

ABR ‚Äî Average Bitrate

Targets an average while varying. Not recommended ‚Äî a mix of CBR and VBR.
